CT-Tutorial

Tutorial code for CT reconstruction with high dynamic range images

Conversion of a CT image to a noisy sinogram and reconstruction with noise and without noise.

This notebook is a tutorial on reconstruction of high-dynamic range CT images with proper physical scaling and a realistic noise model.

Files:

  • CT_Tutorial.ipynb has code to do forward radon projection and reconstruction using filtered backprojection. A key contribution is a description of the units and intensity scaling that appears in real CT images.
  • make_data.ipynb has code to create high dynamic range images that have some properties similar to those from baggage scans.

Outline of processing in CT_Tutorial.ipynb:

  1. Read image: in Hounsfield units (air=0, water=1000)
  2. Scale so that air=0, water=1
  3. Mask for circular ROI
  4. Take the raw sinogram using a specified number of views
  5. Scale to account for pixel pitch and x-ray density
  6. Add noise to the sinogram using the correct variance
  7. Reconstruct from the original and noisy sinogram
  8. Display the reconstructions and sinograms

Examples images and reconstructions, shown at several intensity windows.
